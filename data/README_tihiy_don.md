Annotated "Тихий Дон" (Sholokhov), chapters I-X (Part 1, Book 1) into the speaker encoder dataset.

### Результаты
- **doc**: `tihiy-don-ch1-10`
- **Сегментов**: 642 (261A + 381D)
- **Персонажи**: 27, из них 20 с ≥2 репликами
- **Главные**: григорий (91), пантелей (82), аксинья (46), митька (39)

### Ключевые паттерны атрибуции

1. **Inline-атрибуция через тире**: «— Реплика, — сказал X.» → всегда разбивать на D («Реплика») + A («сказал X»). В тексте Шолохова этот паттерн встречается постоянно.

2. **Диалог без атрибуции**: когда реплики идут подряд с тире, спикер определяется из контекста — чередование между двумя персонажами или продолжение того же.

3. **Атрибуция через «сказал X» после реплики**: «Реплика, — сказал X» → X всегда атрибутирует предыдущую реплику.

4. **Сложный случай**: «— Реплика, — сказал X, — продолжение.» → D + A + D (три сегмента из одной строки).

5. **Кавычки «…» внутри авторского текста**: текст в кавычках → D, остальное → A.

### Проблемы, которых избежать
- Эм-тире в середине реплики (как пауза) НЕ является сменой говорящего: «Глаза тусменныи, — небось, не сладко на чужой сторонушке»
- Одиночные восклицания/междометия в кавычках — не диалог, если нет смены говорящего
- Строки с одним тире «—» (пустой маркер диалога) нужно отбрасывать
- Хор/группа («бабы», «казаки») — допустимый speaker для коллективных реплик
